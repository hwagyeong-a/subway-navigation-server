## 12.4 API 응답 변경

### `/direction` 응답 (변경 후)

```json
POST /direction { "from": "station_exit", "to": "floor1_hall" }
→ 200 { "angle": 268, "cardinal": "W", "clock": 9 }
```

- `angle`: 절대 방위각 (0~360°, 정북=0, 시계방향) — 박경찬 실측값
- `cardinal`: 8방위 식별자 (N/NE/E/SE/S/SW/W/NW)
- `clock`: 시계 방향 (1~12시)

### `/route` 응답 (변경 후)

```json
POST /route { "from": "station_exit", "to": "down_platform" }
→ 200 {
    "path": [
      { "node": "station_exit",         "floor": "ground", "zone": "entrance", "edge_to_next": null    },
      { "node": "floor1_hall",          "floor": "1F",     "zone": "hall",     "edge_to_next": "flat"  },
      { "node": "fare_gate",            "floor": "1F",     "zone": "gate",     "edge_to_next": "flat"  },
      ...
      { "node": "down_platform",        "floor": "B1",     "zone": "platform", "edge_to_next": null    }
    ]
  }
```

- 각 노드에 `floor`, `zone`, `edge_to_next` 추가
- `edge_to_next`: 다음 노드까지의 이동 형태 (`flat` / `stairs` / `branch` / `null`(마지막))
- `DANGER_DESTINATION` 에러 코드 제거
