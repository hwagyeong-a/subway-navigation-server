## 📝 작성 규칙

- **헤딩**: H1은 문서 제목, H2는 섹션 번호 포함(`## 2.1 사용자 정의`)
- **다이어그램**: Mermaid 사용 (GitHub에서 자동 렌더링되며, 보고서에는 스크린샷으로 활용 가능)
- **인용**: 원본 요구사항 문서에서 그대로 가져온 부분은 `>` blockquote로 표시
- **언어**: 본문 한국어, 코드/식별자는 영어
- **표**: 비교/요약 정보는 표로 정리

## 📖 보고서 작성 시 활용 방법

1. **계획서/제안서 작성 시**: 01, 02, 03, 04 문서 우선 참조
2. **설계 문서 작성 시**: 03, 04, 05, 06 문서 활용
3. **개발 문서 작성 시**: 05, 06, 07 문서 활용
4. **중간 발표 시**: 03, 07, 08 문서 활용
5. **최종 발표 시**: 전 문서 + 09(리스크 대응 결과) 활용
